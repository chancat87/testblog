# 关于本站

import { Metadata } from "~/components/metadata/metadata";

<Metadata author="swwind" time={1677859200000} />

本站建立于癸卯年二月十三，使用 Qwik City 框架搭建网站引擎，并且托管于 Cloudflare Pages。

由于是静态网站，所以评论功能什么的全都被我砍掉了，现在这个网站什么东西都没有剩下了。<s>过去的文章懒得迁移了，就这样摆烂其实我感觉也不错。</s>待到 Qwik City 的静态网站成熟了我再尝试迁移过去。之前的博客里面大部分都是我自己都不想看的黑历史文章，所以我决定重新开一个博客，专心写一些技术类的文章，或者闲聊的文章。

网站的样式大部分是抄袭 Telegraph 的，我现在觉得大圆角什么的还是不如极简风有意思。SEO 懒得做了，反正也没有人看，会来看我博客的估计也都认识我。为了专心于阅读，因此网站摒弃了大部分的花里胡哨和没用的破烂功能，除了一个 Header 和一个 Footer 就没有其他的装饰内容了。

本站的创建也要感谢以下用户的帮助：

- 感谢阿里云不停地发短信提醒我原来的服务器快要到期了；
- 感谢杜洛夫叔叔的 Telegraph，为我创建这个极简的博客提供了抄袭的范本；
- 感谢 Qwik 开发团队为我们实现了一个如此简洁的全栈框架。

如果您仔细观察，您会发现网站的日期全都是使用农历的日期。这么做的目的是给您造成一些困扰，使您更加关注到文章发布的日期。
